# Sleep Quality and Productivity Predictor

**Веб-приложение с машинным обучением для предсказания продуктивности**

---

##  Выполнение требований задания

-  **Модель ML**: Обучена на табличных данных, бинарная классификация
-  **Веб-приложение**: Gradio интерфейс  
-  **API**: Автоматически создается Gradio для взаимодействия с моделью
-  **Репозиторий**: Полный набор файлов с документацией

##  О проекте

Веб-приложение для предсказания уровня продуктивности на основе параметров сна и образа жизни с использованием машинного обучения.

##  Техническая реализация

**Модель машинного обучения:**
- Алгоритм: Ensemble (Voting Classifier)
- Задача: Бинарная классификация продуктивности
- Точность: > 85%
- Признаки: 21 engineered feature

**Архитектура:**
- Backend: Python + Scikit-learn + Gradio API
- Frontend: Gradio веб-интерфейс

##  Использование

1. Введите параметры сна и образа жизни
2. Получите предсказание с уверенностью модели

**Пасхалки:**
-  Все нули 
-  Возраст 40, работа 12 часов 

## Файлы проекта

- `app.py` - основное приложение (Gradio + ML)
- `requirements.txt` - зависимости Python
- `*.pkl` - обученные модели и препроцессоры
- `*.json` - метрики модели
